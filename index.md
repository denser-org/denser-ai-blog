## Local Development

To preview the site locally with Jekyll:

### Prerequisites
- Ruby 3.2+ (install via Homebrew on macOS: `brew install ruby`)
- Bundler gem: `gem install bundler`

### Setup and Run
1. **Clone the repository**:
   ```bash
   git clone https://github.com/denser-org/denser-ai-blog.git
   cd denser-ai-blog
   ```

2. **Install dependencies**:
   ```bash
   bundle install
   ```

3. **Start the local server**:
   ```bash
   bundle exec jekyll serve --livereload
   ```

4. **View the site**: Open `http://localhost:4000` in your browser

The site uses the **Jekyll Slate theme** for a dark, wide-layout design optimized for technical content.

### Live Editing
With `--livereload`, any changes to markdown files or `_config.yml` will automatically refresh in your browser.
